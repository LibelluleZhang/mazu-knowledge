27 · 后期工作流

AI生成的素材是「原料」，后期是把原料变成「作品」的炼金术。统一调色、节奏剪辑、字幕规范、双版本输出。

一、素材管理规范

文件夹结构（每集一个）


EP01_渔火/
├── 01_AI原始素材/        # Seedance/ComfyUI直出，不可修改
│   ├── C01_海岸黎明.mp4
│   ├── C02_庙内供香.mp4
│   └── ...
├── 02_精选素材/          # 导演筛选后的可用镜头
├── 03_调色后/            # 统一LUT后的素材
├── 04_声音/              # 配音/音效/配乐
│   ├── VO/               # 配音文件
│   ├── SFX/              # 音效
│   └── BGM/              # 配乐
├── 05_字幕/
├── 06_工程文件/          # DaVinci/剪映项目文件
└── 07_输出/              # 最终成片
    ├── EP01_竖屏_1080x1920.mp4
    └── EP01_横屏_1920x1080.mp4

命名规范

  • 镜头：EP{集号}_C{镜号}_{简短描述}.mp4
  • 版本：EP01_v1.0_内部审阅.mp4 → EP01_v1.2_终稿.mp4

    • 二、剪辑组织原则

    节奏控制

  • 短剧黄金法则：前3秒必须抓住注意力
  • 每集3-5分钟，平均镜头长度3-5秒
  • 情绪高潮处镜头可以拉到8-10秒（慢镜头抒情）
  • 避免连续3个以上AI空镜（会暴露AI感）

    • AI素材特有的剪辑技巧

  • 交叉剪辑：AI空镜（海浪）→ 实拍/半实拍人物反应 → AI空镜（香火）——用人物镜头「打断」AI感
  • 叠化过渡：AI镜头之间用0.5-1秒叠化替代硬切，减少跳跃感
  • 遮罩转场：利用烟雾/海浪/黑场做自然过渡
  • 变速：AI出片帧率不稳的镜头，后期做光流法补帧或慢放

    • 三、调色统一方案

    主LUT：「妈祖色」

    
基调：暖黄+青灰（南国海洋的温润感）
色温：5800K-6200K（略偏暖）
对比度：中等偏低（保留暗部细节，适合手机屏幕）
饱和度：-10%（降饱和增加古朴感）
暗角：轻微（集中视线）

    场景LUT变体

    场景类型调性要点
    庙内祭祀暖金+暗红烛光感，暗部不压死
    海洋白天青蓝+淡金天空不过曝，海水有层次
    海洋夜晚/风暴墨蓝+银灰保留浪花白色细节
    回忆/梦境褪色+柔光加film grain模拟记忆质感
    妈祖显灵金辉+柔焦神圣但不廉价

    操作流程（DaVinci Resolve）

  • 所有素材导入后，先套「妈祖色」主LUT
  • 镜头匹配：选一个「标准镜头」做参考，逐一匹配色温/曝光
  • 肤色保护：如有实拍人物，用qualifier单独处理肤色
  • 输出前做最终通调：全片播放检查色温一致性

    • AI素材特殊处理

  • AI视频常有色温漂移（同一prompt不同seed出图色温不同）
  • 必须逐镜头检查，必要时逐镜头调白平衡
  • 建议在拍摄（生成）阶段就用固定seed+参考图减少色温差

    • 四、字幕规范

    中文字幕

  • 字体：思源黑体 Medium / 阿里巴巴普惠体（免费可商用）
  • 字号：竖屏48px，横屏36px（以1080p为基准）
  • 颜色：白色+黑色描边（2px）
  • 位置：竖屏距底部15%，横屏距底部8%
  • 每行不超过18字
  • 对白字幕+旁白字幕用不同样式区分（旁白用斜体或灰色）

    • 英文字幕

  • 字体：Source Sans Pro / Inter
  • 字号：竖屏28px，横屏22px
  • 颜色：浅黄（#F5E6C8）匹配国风暖色调
  • 位置：中文下方

    • 字幕工具

  • 剪映：自动语音识别+双语字幕（中文较准）
  • ArcTime：专业字幕软件，支持双语时间轴
  • AI翻译：DeepL翻译字幕（文学性对白需人工润色）

    • 五、输出规格

    竖屏版（抖音/小红书/视频号）

    
分辨率：1080×1920（9:16）
编码：H.264 / H.265
码率：15-20 Mbps（VBR）
帧率：24fps / 25fps
音频：AAC 320kbps
时长：3-5分钟
文件名：EP{集号}_{标题}_竖屏_v{版本号}.mp4

    横屏版（B站/YouTube/备份）

    
分辨率：1920×1080（16:9）
编码：H.264 / H.265
码率：25-35 Mbps（VBR）
帧率：24fps / 25fps
音频：AAC 320kbps
文件名：EP{集号}_{标题}_横屏_v{版本号}.mp4

    竖屏拍摄建议（重要！）

  • AI生成时直接按720×1280竖屏构图
  • 不要先做横屏再裁剪——构图会崩
  • 关键元素放在画面中间60%区域（保证横屏裁剪后可用）

    • 六、审片流程

    
内部V1 → 导演审 → V2修改 → 编剧/美术审 → V3精修 → 全组试映 → 终稿

    每轮修改不超过2天。V3之后只做微调不做大改。
