28 · 故事种子

不是完整剧本，而是「如果这样……会怎样？」的火种。按难易度标注，团队可以直接选用。每个种子一句话梗概 + 情感类型 + 建议集数。

难度说明

  • ★☆☆ 简单：场景少、角色少、AI擅长镜头多。适合开篇试水。
  • ★★☆ 中等：多场景/多角色/需精细情感。适合主力剧集。
  • ★★★ 困难：复杂叙事/大量人物特写/历史考据。适合团队成熟后挑战。

    • 故事种子

    1. 【★☆☆】最后一炷香

    梗概：一个老渔民每次出海前都给妈祖烧一炷香。他知道这炷香不能平息风暴，但他还是烧了——这是他跟自己对话的方式。这一次，是他最后一次出海。

    情感：告别 / 信仰 / 克制深情

    集数：1集（3-4分钟）

    2. 【★☆☆】四十年汇款

    梗概：马来西亚一个华侨家庭，每年除夕往湄洲岛一座小庙汇一笔钱，不多不少，刚好够庙里一年的香火。40年没断过。重孙女翻出汇款单，决定飞回湄洲岛看看那座庙。

    情感：乡愁 / 传承 / 寻根

    集数：2集（每集4分钟）

    3. 【★★☆】分灵的重量

    梗概：台湾某妈祖庙的「分灵」要回湄洲岛「谒祖」。护送分灵神像的是一个从来没去过大陆的年轻人。在海上，他做了一个梦。

    情感：身份认同 / 血脉 / 成长

    集数：3集

    4. 【★★☆】退潮时出现的石像

    梗概：湄洲岛一个不起眼的海蚀洞里，只有大退潮时才能看到一尊石雕妈祖像。一个研究海平面变化的博士生发现了它，也发现了一段被潮水淹没的往事。

    情感：发现 / 历史 / 惊鸿一瞥

    集数：2-3集

    5. 【★☆☆】渔网里的妈祖令旗

    梗概：新渔民用拖网捞上来一面褪色的妈祖令旗。老渔民说这是几十年前某艘遇难渔船留下的，令旗还在替那个人守护这片海。

    情感：悬疑 / 敬畏 / 因果

    集数：1集

    6. 【★★☆】台风夜的值守人

    梗概：台风登陆湄洲岛那夜，祖庙的管理员没有撤离。他要守着千年不灭的香火。天亮时风暴过去，他发现妈祖像面向的方向变了。

    情感：坚守 / 灵异 / 孤勇

    集数：2集

    7. 【★★★】沈有容与妈祖

    梗概：1604年，明代将领沈有容收复澎湖，据传得妈祖神助。我们不讲战争，只讲战前一晚，沈有容独自在海边坐了一夜。他在想什么？

    情感：历史 / 沉思 / 家国

    集数：3-4集（需历史考据，AI难以还原明代军备）

    8. 【★☆☆】沙滩上的名字

    梗概：退潮后，沙滩上出现用贝壳摆成的两个名字。涨潮时被冲掉。第二天退潮，名字又出现了。谁每天晚上来摆？为什么是两个名字？

    情感：悬念 / 深情 / 守护

    集数：1-2集

    9. 【★★☆】渡海来的人

    梗概：一个中年人从福建偷渡去东南亚（设定在百年前）。蜷在船舱最底层，怀里抱着一个巴掌大的妈祖像。到了彼岸，他把妈祖像放在神龛上，再没回过家乡。

    情感：漂泊 / 移民 / 信仰的力量

    集数：2-3集

    10. 【★★☆】庙口的猫

    梗概：以一只祖庙里的流浪猫为主视角，看人来人往：求平安的、还愿的、哭的、笑的、独自坐一下午的。猫什么都懂，猫什么都不说。

    情感：温情 / 日常 / 众生相

    集数：单集or系列（可做成松散集锦）

    11. 【★★★】两座妈祖庙之间

    梗概：湄洲祖庙和鹿港天后宫之间有某种「感应」——两地同时出现的异常天象，指向一段被刻意遗忘的历史。一个记者和一位庙祝联手追查。

    情感：悬疑 / 历史 / 两岸

    集数：4-5集（长线叙事）

    12. 【★☆☆】还愿

    梗概：一个企业家多年前在妈祖面前许愿「如果公司上市就捐一座庙」。他真的上市了，也真的来还愿。但还愿的时候，他说了另一个故事。

    情感：反转 / 人性 / 信仰的秤

    集数：1集

    使用建议

  • 第一周：选用2个★☆☆种子做测试，跑通23-AI短剧工作流全流程
  • 第二周起：★★☆种子进入主力排期
  • 第三周后：视团队磨合情况选1个★★★种子做「精品集」
  • 每个种子可先拍1集试播，好评再加集（「剧集开发式」创作）
