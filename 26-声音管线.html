26 · 声音管线

没有声音的短剧是没有灵魂的。妈祖故事的声音世界：闽南语的潮汐、南音的幽咽、海风的叹息。AI生成+人工精修。

一、配音方案：闽南语TTS

主方案：ChatTTS / CosyVoice + 闽南语微调

  • 基座模型：CosyVoice 2.0（阿里，支持few-shot音色克隆）
  • 微调策略：收集3-5分钟闽南语母语者录音作为参考音色
  • 目标：至少覆盖妈祖（女声·温厚慈悲）、老渔民（男声·沧桑）、年轻信众（男/女）

    • 备选方案：人工配音 + AI增强

  • 如果TTS闽南语效果不达预期，采用人工录音
  • 用AI做降噪、去齿音、EQ均衡
  • 工具：Adobe Podcast Enhance / Resemble AI

    • 口型同步（Lip Sync）

  • MuseTalk：输入视频+音频，驱动口型。实测中文口型效果一般，闽南语更未知
  • 策略：尽量避开正面说话镜头。用侧面、背影、环境遮挡来规避口型问题
  • 必须正面说话时：后期单独处理，用Wav2Lip手动修

    • 二、环境音效AI生成

    分层音效架构

    
第1层 · 环境底噪（全程贯穿）
├── 海浪声（根据不同场景：平静/汹涌）
├── 风声（海风/穿堂风/竹林风）
├── 庙宇氛围（远处诵经/木鱼/钟声）
└── 渔港环境（船发动机/缆绳/吆喝声）

第2层 · 动作音效（同步画面）
├── 脚步（沙滩/石板/木船板）
├── 器物（香炉放置/灯笼点燃/渔网撒开）
└── 衣物（绸缎摩擦/蓑衣抖动）

第3层 · 情绪音效（关键时刻）
├── 心跳声
├── 钟鸣（妈祖显灵时刻）
└── 寂静（留白也是一种声音）

    AI音效工具

  • ElevenLabs SFX：文字描述生成音效（如「木船在波浪中摇晃的吱嘎声」）
  • Stable Audio：长音频环境生成
  • AudioLDM 2：开源方案，可本地部署

    • 音效库建设（同步进行）

  • 实地录制：湄洲岛海岸、祖庙内、渔港码头（派1人专门采音）
  • 整理归档：按场景/情绪/器物三级分类
  • 命名规范：场景_类型_描述_时长.wav（如 渔港_环境_清晨码头_120s.wav）

    • 三、配乐风格指南

    核心方向：南音采样 + 现代电子融合

    南音采样素材：

  • 琵琶（南琶，横抱演奏）
  • 洞箫（尺八般幽远）
  • 二弦、三弦
  • 拍板节奏

    • 现代电子层：

  • Ambient pad（氛围铺底）
  • Subtle beat（克制节奏，不喧宾夺主）
  • 低频 drone（海洋的深沉感）

    • 参考艺术家/作品

  • 林海《琵琶相》—— 传统器乐+现代编曲的标杆
  • 坂本龙一《Async》—— 氛围与留白的极致
  • 王勇《往生》—— 民乐与电子融合实验
  • 汉斯·季默《Interstellar》管风琴段落 —— 神圣感的参考

    • AI配乐工具

  • Suno / Udio：输入风格描述生成配乐小样
  • 策略：AI生成 → 人工挑选 → 南音采样替换部分声部 → 混音
  • 注意：AI生成的音乐版权归属不明确，最终成品建议人工重混

    • 四、混音标准

    响度标准（抖音平台）

  • 综合响度：-14 LUFS（与平台推荐一致）
  • 真峰值：不超过 -1 dBTP

    • 分层混音比例

    层次相对音量说明
    对白基准 0dB清晰度为第一优先级
    环境音效-12~-18dB感知到但不抢戏
    动作音效-6~-10dB与画面同步突出
    配乐-15~-20dB情绪烘托，对白时自动闪避(ducking)

    五、声音制作管线排期

    
剧本定稿 → 配音（2天）→ 环境音效（1天，与AI视频同步启动）→ 
配乐（2天）→ 混音（1天）→ 终审

    音频组建议3人：1人配音+口型、1人音效+素材库、1人配乐+混音。

    📚 参考链接

    TTS配音

  • CosyVoice 2.0 GitHub — 阿里通义实验室开源，支持few-shot音色克隆（3-5分钟参考音频即可），推荐用于妈祖角色
  • ChatTTS GitHub — 开源对话式TTS，韵律自然，适合日常对白场景
  • ElevenLabs — 商业级AI语音，29种语言+语音克隆，质量最高但需付费（$5/月起）
  • Resemble AI — AI语音增强+降噪+EQ均衡，人工录音后处理的推荐工具

    • 口型同步

  • MuseTalk GitHub — 腾讯音乐开源，输入视频+音频驱动口型，latentspace方法质量高于Wav2Lip
  • Wav2Lip GitHub — 学术基准口型同步，作为MuseTalk效果不达预期时的手动兜底方案

    • AI音效

  • ElevenLabs SFX — 文字描述生成音效："木船在波浪中摇晃的吱嘎声"→一键生成
  • Stable Audio — Stability AI出品，长音频环境生成（海浪/风声/庙宇氛围2-3分钟连贯环境音）
  • AudioLDM 2 GitHub — 开源音效生成方案，可本地部署避免API费用

    • AI配乐

  • Suno — AI音乐生成首选，输入"南音琵琶+ambient pad+海洋氛围"即可生成配乐小样
  • Udio — AI音乐生成备选，音质优于Suno但生成速度较慢
